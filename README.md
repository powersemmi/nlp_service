# NLP_SERVICE

## Instalation:
For arch:
```bash
# Clone repo
$ git clone git@github.com:powersemmi/nlp_service.git && cd nlp_service
# Install virtualenv
$ sudo pacman -S python-virtualenv
# Create venv
$ python3 -m venv .venv
# Activate venv
$ source .venv/bin/activate
# Install requarements
$ pip install -r req.txt
# Edit configure file
$ vim config.py
# Run app
$ python -m flask run
 * Running on http://127.0.0.1:5000/
```
